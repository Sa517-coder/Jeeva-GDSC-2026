## 🎯 The Problem
**70 crore rural Indians** face **1 doctor per 10,000 people** (WHO standard: 1:1,000). Low-literacy users can't articulate symptoms effectively.

## 💡 Jeeva Solution
**Voice-first AI health assistant** analyzing symptoms → remedies or **108 ambulance dispatch**.

## 📱 LIVE PROTOTYPE FEATURES
| Screen | Functionality |
|--------|---------------|
| **Home** | Voice orb dashboard |
| **Symptoms** | 4x4 tappable icon grid |
| **Results** | AI risk meter + remedies |
| **Emergency** | 108 SOS dispatch |
| **Hospitals** | Google Maps integration |
